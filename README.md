TimeLineStamp is an Obsidian plugin that allows you to timestamp your text/images. These timestamps are then viewable in a timeline window. 

#Installation: 

##Option 1:
1. Clone the repo to your obsidian plugins folder
```sh
git clone https://github.com/Moriz82/TimeLineStamp.git
mv TimeLineStamp /path/2/plugins/folder
```
